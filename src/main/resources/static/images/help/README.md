# 도움말(/help) 스크린샷 이미지

이 폴더에 도움말 페이지에서 사용하는 기능별 스크린샷 이미지를 넣습니다.
`help.html` 의 각 기능 카드는 아래 파일명을 참조합니다. 파일이 없으면
"스크린샷 준비 중" 자리표시자가 자동으로 표시되므로, 준비되는 대로
같은 파일명(.png 권장)으로 이 폴더에 저장하면 즉시 반영됩니다.

| 파일명 | 기능 |
| --- | --- |
| dashboard.png | 로그인 후 첫 화면(대시보드) · 출퇴근 체크 |
| notifications.png | 헤더 알림 |
| ai-chat.png | AI 비서 채팅 |
| profile.png | 내 정보 / 계정 설정 |
| approval.png | 전자결재 작성·상신 |
| approval-template.png | 결재 양식 관리 (권한 필요) |
| calendar.png | 일정(캘린더) 조회 |
| my-attendance.png | 내 근무표 |
| board.png | 게시판 글 읽기·쓰기 |
| board-manage.png | 게시판 관리 (권한 필요) |
| permission.png | 권한 관리 (회사 대표) |
| dept.png | 부서 관리 (권한 필요) |
| position.png | 직급 관리 (권한 필요) |
| waiting.png | 승인 대기 / 가입 승인 (권한 필요) |
| member-org.png | 조직도 구성 · 인사 이동 (권한 필요) |
| attendance-admin.png | 근태 관리 (권한 필요) |
| trash.png | 통합 휴지통 (권한 필요) |
| system-log.png | 시스템 로그 (회사 대표) |

> 권장 사이즈: 가로 1200px 내외, 16:9~16:10 비율. 캡처 시 개인정보(실명/사번/이메일)는 마스킹하세요.
